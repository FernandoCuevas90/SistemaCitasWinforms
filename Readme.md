# Sistema de Citas WinForms

Aplicación de escritorio desarrollada en WinForms para la gestión de citas médicas.

## Funcionalidades

* Registro de citas para estudios:

  * Rayos X
  * Ultrasonido
  * Análisis de sangre
* Edición de citas existentes
* Eliminación de citas
* Búsqueda de citas
* Búsqueda de citas por ID de usuario

## Tecnologías utilizadas

* C#
* Windows Forms
* .NET

## Descripción

Esta aplicación permite gestionar citas médicas de forma sencilla, facilitando el registro, consulta y administración de la información de los pacientes y sus estudios.
